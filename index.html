KeySurfer Getting Started

Thank you for downloading KeySurfer.

This page will quickly teach you how to use KeySurfer.

  1. Press 
    shift+space
    to active the KeySurfer window. It will appear at the bottom of your screen
  2. Type "link" into the KeySurfer search bar.
  3. Notice how it hightlights this link? Press enter to click it.

Good job! Let's learn how to handle multiple links now

  1. Type "click" into the KeySurfer search bar.
  2. Notice how each link starting with click is highlighted?
  3. Use the up/down arrow keys to choose "click3" and then press enter.
  • Click 1
  • Click 2
  • Click 3
  • Click 4

You did it!

If you don't want to use the up/down arrow keys to change elements you can also press alt and then a number to click that labelled link.
